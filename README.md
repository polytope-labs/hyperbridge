# hyperbridge

Hyperbridge: Verifiable interoperability components for cross-chain messaging and asset flows.

TL;DR: A set of interoperable Rust components and documentation showing verifiable cross-chain patterns.

## Features
- Anchor/Rust bridge examples
- Verifiable message routing and proofs
- Examples and docs for testing and deployment

## Quick start
```bash
git clone https://github.com/Ovodo/hyperbridge.git
cd hyperbridge
# build and run tests
cargo test
```
